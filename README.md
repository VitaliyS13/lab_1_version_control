# lab_1_version_control
# Опис проєкту Розрахунково-графічна робота
У репозиторії розміщено навчальний рохрахунково-граічна робота, що виконується у рамках дисципліни «Алгоритміізаціія та програмування» (1 курс). 
Призначення програми – програмування типових оперцій обробки даних представлених одновимірним масивом.
# Основні можливості проєкту
Основні можливі операції над одновимірним масивом:
- введення розмірності масиву
- введення вхідних даних до масиву
- виконання математичних та логічних операцій наж елементами масиву
- виведення елементів масиву для користувача
